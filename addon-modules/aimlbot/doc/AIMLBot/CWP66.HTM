AIMLBot.cNodeMapper Class
Represents an individual node in the AIML "brain" and maps the branches in the Graphmaster structure.

Access: Public
Base Classes: Object
  Members Description  
    htChildren Represents the branches from this nodemapper

 
    thisCategory the Category that is to be returned if this node is matched

 
    sWord the word that identifies this nodemapper at the parent nodemapper (i.e. the nodemappers id within it's parent node)

 
    cNodeMapper    
    evaluate Searches the nodemapper with the input string

 
    addCat Adds a category to the graphmaster structure

 
Remarks:
This class incorporates much of the donkey work of searching the graphmaster structure to the specifications defined at: http://www.alicebot.org/TR/2001/WD-aiml/ Specifically, the precedence and processing of the "_" and "*" wildcards as defined by the AIML standard (referenced above)/.
