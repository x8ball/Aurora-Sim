AIMLBot.cAIMLLoader Class
Takes care of processing and loading AIML files into the graphmaster

Access: Public
Base Classes: Object
  Members Description  
    message Holds the message the the cAIMLLoader returns to the bot that calls it for inclusion in the bot's logfile

 
    myGraphMaster represents the Graphmaster to be populated by the AIML files

 
    counter counts the number of categories added to the bot's brain

 
    processtopic processes "topic" nodes and adds any categories found as children into the graphmaster's set of nodes

 
    processcategory processes "category" nodes into cCategory objects for the graphmaster

 
    processcategory processes "category" nodes into cCategory objects for the graphmaster topic tag

 
    addfile Processes the categories found within a certain file

 
    processdirectory Given a directory path get the aiml files and extract the categories from them

 
    cAIMLLoader Doh! This is a constructor - gets the graphmaster brain to be loaded and resets everything to nice values for the logs :-)

 
    loadAIML Given a path to a directory, this method will look for *.aiml files, process them into cCategory objects and add them to the bot's "brain". Returns any messages about the status of the upload into the GraphMaster brain.

 
Remarks:
Used the specification of AIML v.1.0.1 as the basis for storing the various categories. See http://www.alicebot.org/TR/2001/WD-aiml/ for more information.
