AIMLBot.cCategory.alTopicStar System.Collections.ArrayList alTopicStar
NOTA BENE: this attribute is only set by a nodemapper when a category is returned in response to some user input. If the "topic" part of the PATH contains a wildcard then this attribute will contain the block of text (as individual words) that the user has inputted that is represented by the wildcard.
