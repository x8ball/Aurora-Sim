AIMLBot.cProcessor Class
Contains all the methods for processing AIML found in the "template" part of the cCategory returned by the Graphmaster.

Access: Public
Base Classes: Object
  Members Description  
    random deals with the "random" tag.

 
    star deals with the "star" tag.

 
    thatstar deals with the "that" tag.

 
    topicstar deals with the "topicstar" tag.

 
    that deals with the "that" tag

 
    input deals with the "input" tag

 
    getTag deals with the "get" tag

 
    bot deals with the "bot" tag

 
    setTag deals with the "set" tag

 
    upper deals with the "upper" tag

 
    lower deals with the "lower" tag

 
    formal deals with the "formal" tag

 
    formal overloaded "formal" text processing

 
    sentence deals with the "sentence" tag

 
    think deals with the "think" tag

 
    person deals with the "person" tag

 
    person2 deals with the "person2" tag

 
    gender deals with the "gender" tag

 
    srai deals with the "srai" tag

 
    sr deals with the "sr" tag

 
    processtag processes the individual aiml tags found within the "target" portion of a category

 
    process Given a cCategory, returns the same object but with the template section processed and changed appropriately

 
Remarks:
These routines are based upon the AIML specification v1.0.1 that can be found at: http://www.alicebot.org/TR/2001/WD-aiml/
