AIMLBot.cUser Class
Represents a single user. Stores helpful information like a predicate list specific to this user and instantiates a log for the user.

Access: Public
Base Classes: Object
  Members Description  
    sUserID a copy of the sUserID that is the object's key in the Users hashtable in cAIMLBot

 
    alThat the last utterence from the bot to the user

 
    alInput the last input from the user to the bot

 
    sTopic the value of the "topic" predicate

 
    Predicates the predicates associated with this particular user

 
    cUser    
    addlog adds a log message for this particular user

 
    getThat returns the last full reply from the bot as a single string (as oppose to the arraylist that stores it)

 
Remarks:
This is really just a helpful class to have around when administering the system.
