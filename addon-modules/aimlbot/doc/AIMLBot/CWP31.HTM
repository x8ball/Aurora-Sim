AIMLBot.cResponse Class
A simple data class that holds information about the reply from a bot.

Access: Public
Base Classes: Object
  Members Description  
    userID    
    sUserID the userID who caused this response

 
    botID    
    sBotID the botID that generated this response

 
    input    
    sInput the raw input from the user

 
    output    
    alOutput the response from the bot

 
    that    
    sThat the "that" value (previous bot output)

 
    topic    
    sTopic the value of the "topic" predicate

 
    file    
    sAIMLFile the AIML file from where the response was defined (useful for debugging)

 
    cResponse Define the values contained within the cResponse object

 
    getOutput Returns the bot's reply as a string rather than as the ArrayList representation that is stored in the cReply class

 
    getThat Returns an appropriately processed arraylist of the output sentences for the next "that" portion of a conversation
